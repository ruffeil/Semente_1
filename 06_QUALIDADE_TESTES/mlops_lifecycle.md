# Ciclo de Vida de Modelagem (MLOps)
- **Rastreamento de Experimentos:** Registro de Hiperparâmetros e Métricas (R2, F1-Score) de cada execução.
- **Model Registry:** Versão dos modelos treinados (v1.0, v1.1). Um modelo só vira "Produção" se bater o benchmark anterior.
- **Data Drift:** Monitoramento automático para detectar se os novos dados do cliente mudaram muito em relação aos dados de treino.
