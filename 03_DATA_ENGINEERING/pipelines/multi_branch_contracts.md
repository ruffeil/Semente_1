# Contratos de Dados por Ramo (MVP)
- **Varejo:** data_venda (Date), id_produto (String), valor (Float).
- **Logística:** data_saida (Date), data_entrega (Date), distancia (Float).
- **Agro:** id_talhao (String), umidade (Float), ndvi (Float).
- **Saúde:** id_paciente_hash (String), data_admissao (Date), exame (Float).
- **Indústria:** timestamp (DateTime), temperatura (Float), vibracao (Float).
