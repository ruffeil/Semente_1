# Pipeline de Produção SEMENTE_FRAME
Fluxo baseado em práticas de bancos e ambientes regulados:
1. **Ingestão:** Validação de Data Contract.
2. **Limpeza:** Tratamento de nulos e outliers.
3. **EDA:** Análise exploratória automatizada.
4. **Estatística:** Testes de hipótese e significância.
5. **Feature Eng:** Criação de variáveis reprodutíveis.
6. **Treinamento:** AutoML com rastreamento de experimentos.
7. **Validação:** Gates de métricas mínimas (Go/No-Go).
8. **Insights:** Relatório executivo via LLM.
