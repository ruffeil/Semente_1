# Padrões de Desenvolvimento
- **Clean Code:** Uso obrigatório de Tipagem Estática (Type Hinting) em Python.
- **GitFlow:** Branch 'main' (produção), 'develop' (homologação) e 'feature/*' (novas funcionalidades).
- **Gerenciamento de Dependências:** Uso de ambiente virtual isolado para evitar conflitos de bibliotecas de ML.
- **Documentação:** Uso de Docstrings em todas as classes para geração automática de manuais técnicos.
